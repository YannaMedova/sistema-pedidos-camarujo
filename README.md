<p align="center">
  Sistema desktop para controle de pedidos, precificação e exportação de planilhas da Camarujo Aquarismo.
</p>

---

## ✨ Funcionalidades

| Função                   | Descrição                                             |
| ------------------------ | ----------------------------------------------------- |
| 📥 Importação            | Importa planilhas de fornecedores (.xlsx, .xls, .csv) |
| 💰 Preço Sugerido        | Cálculo automático baseado na margem                  |
| ✏️ Venda Manual          | Permite ajuste manual dos valores                     |
| ➕➖ Quantidade            | Controle via botões e digitação direta                |
| 📊 Totais                | Cálculo automático de custo, venda e lucro            |
| 💾 Persistência          | Salva automaticamente o último pedido                 |
| 🔄 Novo Pedido           | Limpa tudo e inicia novo processo                     |
| 🔍 Busca                 | Filtro rápido por nome do item                        |
| 🌙 Tema                  | Alternância entre claro e escuro                      |
| 📄 Exportação Cliente    | Planilha formatada para venda                         |
| 📦 Exportação Fornecedor | Planilha organizada para compra                       |
| 🕓 Histórico             | Armazena pedidos fechados                             |
| 🚀 Splash Screen         | Tela inicial com animação                             |

## ▶️ Como Executar

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

2. Execute o sistema:

```bash
python main.py
```
